How to create your Adaptor:¶

At the moment we have the abstract base Adaptor that is describing three abstract sub methods:

  • translate()
  • execute()
  • unbuild()

Translate¶

This method should create a configuration file for your external component, and store it in the output_configs directory located in the package where all the configuration file from the different other Adaptors are. This configuration file should be named after an ID that can be generated with the generator we provide, or another one you’d like. This method return this ID.

Execute¶

This method should execute the wanted commands from the wanted Adaptor. It takes as parameter the ID link to the wanted component to be executed.

Unbuild¶

This method takes as parameter the ID of the wanted component and unbuild it.

Generator¶

We do give access to a method that creates the random ID.
